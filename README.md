# MealMate - AI Powered Recipe and Grocery Helper

An app to simplify

## Description

An App that stores recipes that are created by yourself or imported from the web.

## Tech Stack

- Frontend
  - Vite
  - React
  - Next.js
  - TypeScript
  - Material UI
- Backend
  - Express.js
  - MongoDB
